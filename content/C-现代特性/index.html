C++

C++11

Lambda Expressions

C++11开始支持lambda函数。

Auto类型推导

#include <utility> 引入类型推导，大大方便了泛型编程，解决了类型名过长的问题。

Decltype类型推导

#include <type_traits>

自C++14起可以和auto一起使用。

decltype(auto)

for-each

Range-Based for Loop: Provides foreach-like semantics for iterating through a container or array.

Constexpr

统一初始化语法

nullptr

指针专用的nullptr，解决了NULL和0宏冲突的问题。

Delegating Constructors

Rvalue References

Lvalue: An lvalue (locator value) represents an object that occupies some identifiable location in memory (i.e. has an address).

Rvalue:

New Smart Pointer Classes

C++98定义了一个智能指针auto_ptr（C++17中移除）。
C++11新增了shared_ptr and unique_ptr and weak_ptr。

Threading Library

C++14

Generic Lambdas

Binary literals

二进制支持。
int b = 0b00010000;

decltype(auto)

Variable Templates

C++17

Structured Binding Declarations

Simplify the process of unpacking a tuple, pair, or other aggregate types.

inline variables

Enables inline keyword for variables and allows single definition of global and class static variables in header files.

Nested Namespace

  • Before C++17

  • When C++17

Variable declaration in if and switch

if and switch在C++17中可以在其中进行变量的声明。

Standard FSL

FSL(File System Library)
std::filesystem as a standardized way to manipulate paths, directories, and files.

Standard Library addtions

  • <string_view> (non-owning string reference)
  • <any> (type-safe discrimination union), 
  • <optional> (optional value wrapper), 
  • <variant> (type-safe sum type),
  • <memory_resource> (library for polymorphic allocators).
  • Parallel Algorithms: Adds support for parallel execution of Standard Library algorithms.
